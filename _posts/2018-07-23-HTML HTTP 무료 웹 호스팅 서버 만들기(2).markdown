## 파일질라(Filezilla) 란?

> 일반적으로 웹 호스팅이나 리눅스 서버에 접속하여 파일을 올리고 싶다면 직접 리눅스로 FTP 접속을 통해 파일을 주고 받을 수 있지만, 파일질라를 사용하면 윈도우에서 쉽게 마우스 드래그와 클릭만으로 파일을 수정하고 Upload/Download 가 가능하다.

https://filezilla-project.org/

위 사이트에서 파일질라를 다운받습니다.
