## 리눅스 명령어(Linux Command)

- 파일 인코딩 확인하기
```
$file -bi test.sh
```

- 파일 인코딩 변환하기
```
$iconv -c -f utf-8 -t euc-kr test.sh > test2.sh
```

- 현재경로 부터 찾기
$find -name 'title'

$find -name 'title*'

- 루트경로부터 찾기
$find / -name 'title'

- find / -name '.sh'
