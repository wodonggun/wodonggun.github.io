## 1일차
1. Processor
- CPU(연산장치)
- Process랑 다름(저장소->RAM에 프로그램이 올라간 상태를 Process라고 함)

2. Memory
- RAM 메모리
- HDD,SDD 등 저장소

3. I/O
- 마우스, 키보드, 마우스


```

Fetch = 메인 메모리에 올라온 명령어를 CPU가 한줄 한줄 읽어온다.
Decode = 가져온 명령어를 해석
Execute = 실행



```
